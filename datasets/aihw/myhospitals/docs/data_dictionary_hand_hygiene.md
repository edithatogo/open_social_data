# Data Dictionary for MyHospitals - Hand Hygiene (MYH-HH)

**Dataset Source:** Australian Institute of Health and Welfare (AIHW) - MyHospitals API (`/flat-formatted-data-extract/MYH-HH`)
**Date of Dictionary Creation/Update:** 2025-07-08; reviewed 2026-06-18.

---

## 1. Introduction

This data dictionary describes the variables within the "Hand Hygiene" dataset, specifically from the `MYH-HH` measure category, accessed via the AIHW MyHospitals API's `/flat-formatted-data-extract/` endpoint. The data is processed and stored in this repository in Parquet format.

The purpose of this document is to help users understand the structure, content, and meaning of each variable. The fields are based on the `FormattedDataExtractModel` schema from the API's Swagger definition and observed sample data. The column structure is consistent with other flat formatted data extracts from this API.

## 3. Code Lists / Classifications (Examples for MYH-HH)

*   Example `reported_measure_code`: `MYH-RM0208` ("All audited moments"). Other codes might exist for specific professional groups (e.g., doctors, nurses) or moments of hand hygiene.

### 3.4. `peer_group_code`
