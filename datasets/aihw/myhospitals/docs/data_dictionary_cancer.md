# Data Dictionary for MyHospitals - Cancer Data (MYH-CANCER)

**Dataset Source:** Australian Institute of Health and Welfare (AIHW) - MyHospitals API (`/flat-formatted-data-extract/MYH-CANCER`)
**Date of Dictionary Creation/Update:** 2025-07-08; reviewed 2026-06-18.

---

## 1. Introduction

This data dictionary describes the variables within the "Cancer" dataset, specifically from the `MYH-CANCER` measure category, accessed via the AIHW MyHospitals API's `/flat-formatted-data-extract/` endpoint. The data is processed and stored in this repository in Parquet format.

The purpose of this document is to help users understand the structure, content, and meaning of each variable. The fields are based on the `FormattedDataExtractModel` schema from the API's Swagger definition and observed sample data. The column structure is consistent with other flat formatted data extracts from this API.

## 3. Code Lists / Classifications (Examples for MYH-CANCER)

*This section will be expanded as more specific codelists are analyzed from the data or API metadata endpoints.*

### 3.1. `measure_code` / `reported_measure_code` (within MYH-CANCER)
*   Example `measure_code`: `MYH0001` ("Number of surgeries for malignant cancer").
*   Example `reported_measure_code`: `MYH-RM0004` ("Breast cancer" - disaggregation of MYH0001). Other codes will exist for other cancer types (e.g., Bowel, Lung).

### 3.2. `reporting_unit_code`
*   Codes for hospitals. Example: `H0017` (St Vincent's Hospital [Darlinghurst]).

### 3.3. `reported_measure_category_code` (Disaggregation for MYH-CANCER)
*   Primarily by "Type of cancer". Example: `CANCER_Breast`, `CANCER_Bowel`, `CANCER_Lung`.

### 3.4. `reported_measure_category_type_code`
*   Example: `CANCER_TYPE` (Indicates the disaggregation is by type of cancer).

*Refer to AIHW MyHospitals API metadata endpoints (e.g., `/measures`, `/reported-measures`, `/reporting-units`) for comprehensive lists of codes.*
