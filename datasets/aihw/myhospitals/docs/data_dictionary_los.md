# Data Dictionary for MyHospitals - Length of Stay (MYH-LOS)

**Dataset Source:** Australian Institute of Health and Welfare (AIHW) - MyHospitals API (`/flat-formatted-data-extract/MYH-LOS`)
**Date of Dictionary Creation/Update:** $(date +"%Y-%m-%d")

---

## 1. Introduction

This data dictionary describes the variables within the "Length of Stay" (LOS) dataset, specifically from the `MYH-LOS` measure category, accessed via the AIHW MyHospitals API's `/flat-formatted-data-extract/` endpoint. The data is processed and stored in this repository in Parquet format.

The purpose of this document is to help users understand the structure, content, and meaning of each variable. The fields are based on the `FormattedDataExtractModel` schema from the API's Swagger definition and observed sample data. The column structure is consistent with other flat formatted data extracts from this API.

## 3. Code Lists / Classifications (Examples for MYH-LOS)

*This section will be expanded as more specific codelists are analyzed from the data or API metadata endpoints.*

### 3.1. `measure_code` / `reported_measure_code` (within MYH-LOS)
*   Example `measure_code`: `MYH0014` ("Average length of overnight stays").
*   Example `reported_measure_code`: `MYH-RM0128` ("Cellulitis" - as a condition for which LOS is measured). Other codes will exist for other conditions/procedures.

### 3.2. `reporting_unit_type_code`
*   Example: `H` (Hospital).

### 3.3. `reported_measure_category_code` (Disaggregation for MYH-LOS)
*   Typically by condition or procedure (AR-DRG related). Example: `LOS_Condition_Cellulitis`.

### 3.4. `reported_measure_category_type_code`
*   Example: `AR-DRG` (Australian Refined Diagnosis Related Groups).

*Refer to AIHW MyHospitals API metadata endpoints (e.g., `/measures`, `/reported-measures`, `/reporting-units`) for comprehensive lists of codes.*
