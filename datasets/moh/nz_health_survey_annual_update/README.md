# Annual Update of Key Results: New Zealand Health Survey

**Source:** New Zealand Ministry of Health  
**Original Source Link:** https://www.health.govt.nz/publications/annual-update-of-key-results-202425-new-zealand-health-survey  
**Date Accessed/Processed:** 2026-06-23

## Description

This pack documents the Ministry of Health annual New Zealand Health Survey key results. It broadens repository coverage beyond national statistics agencies and AIHW while keeping to public, reusable social data.

## Data Structure & Access

* **Expected source format:** CSV downloaded from the Annual Data Explorer.
* **Processed format:** Parquet in `data/`.
* **Script:** `scripts/fetch_nz_health_survey_annual_update.py`

Example:

```bash
python datasets/moh/nz_health_survey_annual_update/scripts/fetch_nz_health_survey_annual_update.py --url "path-or-url-to-annual-data-explorer-export.csv"
```

## Methodology & Collection

The Ministry of Health publishes annual key results from the New Zealand Health Survey. The 2024/25 publication states that data can be downloaded from the Annual Data Explorer as CSV and provides methodology links for collection and survey content.

## Known Issues

Annual Data Explorer CSV URLs can be generated by the interactive explorer. This pack keeps the fetch script URL-driven rather than hard-coding an interface-generated export URL.
